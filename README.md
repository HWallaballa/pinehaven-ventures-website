# Pinehaven Ventures Website

This repository contains the Pinehaven Ventures marketing site and venture detail pages built with Next.js (App Router) and Tailwind CSS.

## Local Development

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Key Routes

- `/` - Pinehaven Ventures homepage
- `/ventures` - Ventures overview page
- `/ventures/powerpmis` - PowerPMIS MVP + full platform detail page
- `/ventures/cryptotransactionlog` - CryptoTransactionLog demo-style detail page

## Validation

Run checks before pushing:

```bash
npm run lint
npm run build
```
